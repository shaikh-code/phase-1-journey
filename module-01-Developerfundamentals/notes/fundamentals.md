# How Web Works

Client → sends request → Server → sends response

HTTP Methods:
- GET → fetch data
- POST → send data
- PUT → update
- DELETE → remove

Status Codes:
- 200 → success
- 404 → not found
- 500 → server error


# HTTP & JSON

Request = URL + Method + Headers + Body  
Response = Status + Data

JSON Example:
{
  "name": "Moiz",
  "role": "AI Engineer"
}
